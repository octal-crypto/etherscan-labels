Etherscan Label API

(Since Etherscan doesn't expose one)



List all labels:
octal.art/etherscan-labels/labels.json


Label -> addresses:
octal.art/etherscan-labels/labels/aave.json


Address -> labels:
octal.art/etherscan-labels/addresses/0xffc97d72e13e01096502cb8eb52dee56f74dad7b.json




Label schema:
{                       
  "Label": "",          
  "Description": "",    
  "Addresses": {        
    "": {               
      "Name Tag": "",   
      "Subcategory": "",
      "Token Name": ""  
    },                  
    ...                 
  }                     
}                       



Address schema:
{                       
  "Address": "",        
  "Labels": {           
    "": {               
      "Name Tag": "",   
      "Description": "",
      "Subcategory": "",
      "Token Name": ""  
    }                   
  }                     
}
